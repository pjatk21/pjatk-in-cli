# PJATK in CLI

## Install

### Requirements

 - Node 17, probably 16 works too

### Unix based OSes (macOS/Linux)

```bash
sudo npm i -g pjatk-in-cli
```

> note: if you use nvm, you don't have to pass sudo

### Windows

```pwsh
npm i -g pjatk-in-cli
```

## How to use it

### First run

```bash
pjcli init
```

### Getting latest schedule

```bash
pjcli today
```
